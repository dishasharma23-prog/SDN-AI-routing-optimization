# Sprint Topology (ISP Network)

This setup simulates a real-world ISP topology with 11 nodes using Mininet.
Topology is referred from https://ieeexplore.ieee.org/stamp/stamp.jsp?tp=&arnumber=8530994&isnumber=8530880



## Run
sudo python3 sprint_topology.py

## Features
- OSPF
- RIP
- Fuzzy logic routing
- Security-aware path deviation
