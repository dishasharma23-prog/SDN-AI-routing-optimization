# Sprint Topology (ISP Network)

This setup simulates a real-world ISP topology with 11 nodes using Mininet.
Topology is referred from A. Al-Jawad, P. Shah, O. Gemikonakli and R. Trestian, "Policy-based QoS Management Framework for Software-Defined Networks," 2018 International Symposium on Networks, Computers and Communications (ISNCC), Rome, Italy, 2018, pp. 1-6, doi: 10.1109/ISNCC.2018.8530994. keywords: {Quality of service;Monitoring;Streaming media;Loss measurement;Network topology;Topology;Artificial neural networks;SDN;Policy-Based Network Management;QoS;Neural Network},



## Run
sudo python3 sprint_topology.py

## Features
- OSPF
- RIP
- Fuzzy logic routing
- Security-aware path deviation
